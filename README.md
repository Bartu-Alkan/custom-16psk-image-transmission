# custom-16psk-image-transmission
A MATLAB digital communication project that transmits a grayscale image using a custom 16-PSK-like constellation over an AWGN channel and evaluates BER performance.

Requirements:

- MATLAB
- Signal Processing Toolbox
- Image Processing Toolbox

Input:

- cameraman.tif or another grayscale image

Main features:

- Custom non-uniform 16-PSK-like constellation
- Anonymous fixed random seed
- Gray coding
- Passband modulation
- Coherent demodulation
- Minimum-distance decoding
- BER simulation
- Image reconstruction at different SNR values
